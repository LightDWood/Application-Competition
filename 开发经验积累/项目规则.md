# 项目规则 - AI 创意大赛网站

## 文件创建

必须使用 PowerShell 命令行，不要使用 Write 工具

原因：
- Write 工具可能报告成功但实际未创建文件
- Write 工具对绝对路径解析存在问题
- .trae 目录被系统列入黑名单，禁止写入

正确做法：
1. 使用 Here-String 包裹多行内容
2. 使用 Out-File 命令创建文件
3. 必须指定 UTF8 编码
4. 创建后必须验证

## 命令格式规范

1. 多行文本处理：必须使用 Here-String 包裹
2. 命令分隔：使用分号分隔多个命令，不要使用逗号
3. 路径处理：优先使用相对路径
4. 编码指定：必须指定 UTF8 编码

## 关键教训

1. 不要盲目相信工具返回，必须进行二次验证
2. 优先使用底层命令（PowerShell命令优于高级工具）
3. 路径问题要谨慎，优先使用相对路径
4. 避免在 .trae、.vscode、.git 目录操作文件
