1. 部署前确保本地代码已提交并推送到 GitHub
2. 使用一键部署脚本前确认服务器连接正常
3. 部署后检查 PM2 进程状态和日志

### 11.3 故障排查

1. 前端无法访问 → 检查 Nginx 配置和状态
2. 后端接口报错 → 检查 PM2 进程和日志
3. 数据库连接失败 → 检查 MySQL 服务和网络
4. 部署失败 → 检查 Git 仓库和权限
