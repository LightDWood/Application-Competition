## 📋 问题概述

根据用户反馈，系统在对话中存在意图识别错误，导致 AI 回复不符合预期。

### 用户反馈的问题场景

1. **用户问**: "你有什么能力"  
   **实际回复**: "您好！我是 AI 助手，很高兴为您服务。请问有什么可以帮助您的吗？"（默认问候语）  
   **预期回复**: 应该介绍 AI 的功能和能力

2. **用户说**: "我想做一个咨询订阅的工具"  
   **实际回复**: "您好！我是需求分析助手，很高兴为您服务。请问有什么可以帮助您的吗？"（默认问候语）  
   **预期回复**: 应该进行需求分析，提供 5W2H 框架分析

---

## 🔍 根因分析

### 问题 1: 问答意图识别不全

**原因**: `intentRecognizer.js` 中的 QA 意图模式列表不完整，缺少对"能力"相关问题的匹配。

**问题代码**:
```javascript
QA: [
  /什么 (模型 | 技术 | 功能|AI|系统 | 工具)/i,  // ❌ 缺少 "能力"
  // ... 其他模式
]
```

**影响**:
- "你有什么能力" 无法匹配 QA 模式
- "你能做什么" 无法匹配 QA 模式
- "介绍一下你自己" 无法匹配 QA 模式
- 所有询问 AI 能力的问题都被误判为闲聊

### 问题 2: 需求意图识别不全

**原因**: `intentRecognizer.js` 中的 REQUIREMENT 意图模式列表不完整，缺少口语化表达和行业术语。

**问题代码**:
```javascript
REQUIREMENT: [
  /我要做/,
  /我想 (做 | 实现 | 开发 | 创建)/i,  // ❌ 缺少 "弄"、"搞"、"整" 等口语
  /需要 (一个 | 个|套|款)/i,  // ❌ 缺少 "种"
  // ... 缺少 "咨询"、"订阅"、"CRM" 等行业术语
]
```

**影响**:
- "我想做一个咨询订阅的工具" 无法匹配 REQUIREMENT 模式
- "帮我弄个网站" 无法匹配 REQUIREMENT 模式（"弄" 不在列表中）
- "想要一个 CRM 系统" 无法匹配 REQUIREMENT 模式

---

## ✅ 修复方案

### 修复 1: 扩展 QA 意图模式

**文件**: `server/services/intentRecognizer.js`

**新增模式**:
```javascript
QA: [
  /什么 (模型 | 技术 | 功能|AI|系统 | 工具 | 能力)/i,  // ✅ 添加 "能力"
  // ... 原有模式
  // ✅ 新增：能力相关问答
  /能做什么/i,           // "你能做什么"
  /会什么/i,             // "你会什么"
  /能力/i,               // "你有什么能力"
  /功能.*吗/i,           // "有什么功能吗"
  /介绍.*自己/i,         // "介绍一下你自己"
  /擅长/i,               // "你擅长什么"
  /本领/i,               // "你的本领是什么"
  /本事/i,               // "你有什么本事"
  /如何/i                // "如何使用"
]
```

### 修复 2: 扩展 REQUIREMENT 意图模式

**文件**: `server/services/intentRecognizer.js`

**新增模式**:
```javascript
REQUIREMENT: [
  // ✅ 扩展口语化表达
  /我想 (做 | 实现 | 开发 | 创建 | 弄 | 搞 | 整)/i,
  /需要 (一个 | 个|套|款|种)/i,
  /帮我 (做 | 写 | 实现 | 开发 | 弄 | 搞)/i,
  /想要 (一个 | 个 | 种 | 套 | 款)/i,
  
  // ✅ 新增口语词
  /弄个/i,               // "帮我弄个网站"
  /搞个/i,               // "搞个 APP"
  /整个/i,               // "整个系统"
  /做个/i,               // "做个小程序"
  
  // ✅ 新增行业术语
  /咨询.*工具/i,         // "咨询订阅的工具"
  /咨询.*系统/i,
  /订阅.*系统/i,
  /订阅.*工具/i,
  /管理.*系统/i,         // "合同管理系统"
  /管理.*平台/i,
  /CRM.*系统/i,          // "CRM 系统"
  /CRM/i,                // "想要一个 CRM"
  /小程序/i,             // "电商小程序"
  /APP/i,                // "购物 APP"
  /网站/i,               // "公司网站"
  /平台/i,
  /软件/i,
  /应用/i
]
```
