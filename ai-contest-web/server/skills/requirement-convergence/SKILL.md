# Requirement Convergence Skill（需求收敛技能 v3.0）

## 核心理念

- 🎯 **业务聚焦**：只讨论5个业务维度，用户无需参与技术讨论
- 🔍 **智能收敛**：动态分析上下文，按需追问模糊点
- ⚡ **高效输出**：会话精简（≤200字/轮），完整文档归档
- 🤖 **技术自治**：AI基于业务需求自动推荐技术方案

## 触发条件

### ✅ 应该触发
- 用户说"我想做"/"需要开发"/"规划一个"
- 用户需求模糊不清
- 用户说"帮我理一下需求"
- 用户讨论功能但没想清楚细节

### ❌ 不应触发
- 用户明确要求写代码/实现功能
- 用户已有清晰的需求文档
- 用户只是闲聊

## 约束规则

1. **轮次控制**：每轮只问1个核心问题
2. **选项限制**：最多4个选项，描述≤30字
3. **推荐标记**：🌟标记推荐选项
4. **进度显示**：━━━━━━━━━━━━━░░░░░ 40%
5. **跳过机制**：始终提供"其他/跳过"选项
6. **模糊容忍**：支持 1/A/[A]/直接输入 等多种格式
