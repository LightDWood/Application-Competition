## 风险预警

1. **Task 1**: Markdown 库可能增加包体积
   - 缓解：使用 CDN、按需加载

2. **Task 2**: 流式输出问题可能涉及网络层
   - 缓解：充分测试、添加降级方案

3. **Task 3**: 案例收集工作量大
   - 缓解：优先收集高频场景案例

4. **Task 5**: 外部 API 可能不稳定
   - 缓解：添加超时、重试、降级机制
