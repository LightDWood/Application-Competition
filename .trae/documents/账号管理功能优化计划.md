# 账号管理功能优化计划

## 需求分析

用户要求：
1. 账号管理中要能够展示账号及密码
2. 去掉"查看文档"的按钮

### 关于密码显示的说明

**重要**：系统中密码是以哈希值（password_hash）形式存储的，无法逆向获取原始密码。

可选方案：
- **方案A**：显示默认初始密码（如 admin123），适用于系统预设账号
- **方案B**：添加"重置密码"功能，管理员可重置用户密码为默认值
- **方案C**：显示密码字段但不显示实际密码（显示为 `******`），仅作占位

建议采用 **方案A + 方案B 组合**：
- 对于系统预设账号显示默认密码提示
- 提供"重置密码"按钮，重置为默认密码

### 步骤 2: 添加密码显示列（显示默认密码提示）

**文件**: `src/views/TrainingAdmin.vue`

在用户信息区域添加密码显示：

```vue
<p class="user-meta">
  <span>邮箱：{{ user.email }}</span>
  <span>密码：{{ getDefaultPassword(user) }}</span>
  <span>角色：
    <span :class="['role-badge', user.role]">
      {{ user.role === 'admin' ? '管理员' : '用户' }}
    </span>
  </span>
  <span>注册时间：{{ formatDate(user.createdAt) }}</span>
</p>
```

添加获取默认密码的方法：

```javascript
const getDefaultPassword = (user) => {
  if (user.username === 'admin') {
    return 'admin123 (默认)'
  }
  return '******'
}
```
